## 📌 Sobre o Projeto

Este projeto consiste em uma **Agenda de Contatos desenvolvida em Python**, executada em ambiente **Google Colab**.

A aplicação permite **gerenciar contatos diretamente pelo terminal**, possibilitando:

* Adicionar contatos
* Visualizar contatos cadastrados
* Editar informações
* Remover contatos

Os dados são armazenados em um **arquivo JSON**, permitindo salvar e carregar os contatos da agenda.

Este projeto foi desenvolvido com o objetivo de **praticar lógica de programação, manipulação de arquivos e estruturas de dados em Python**.

---

## 🚀 Funcionalidades

✔️ Adicionar novos contatos

✔️ Listar contatos cadastrados

✔️ Editar informações de um contato

✔️ Remover contatos da agenda

✔️ Salvar dados em arquivo `.json`

✔️ Carregar dados salvos

## ▶️ Como Executar o Projeto

### Opção 1 — Google Colab

1. Abra o **Google Colab**
2. Crie um novo notebook
3. Cole o código da agenda
4. Execute a célula para iniciar o programa

---

### Opção 2 — Executar localmente

Caso queira rodar no seu computador:

```bash
python agenda_de_contatos.py
```

## 💾 Estrutura do Contato

Cada contato é armazenado da seguinte forma:

```json
{
  "nome": "Maria",
  "telefone": "11999999999",
  "email": "maria@email.com"
}
```
